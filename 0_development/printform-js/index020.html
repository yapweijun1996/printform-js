Purchase Terms and Conditions (PTAC)

This PTAC-only demo is used to validate behavior when there are no .prowitem rows. PTAC content is expanded into .ptac-rowitem blocks and remains part of the main pagination flow, so it should start on page 1 with header/docinfo present.

Scope. PTAC paragraphs describe contractual terms and are treated as standard body rows. They follow rowheader repetition rules governed by data-repeat-ptac-rowheader. Because PTAC is not rendered in a separate pass like PADDT, it should not force a new page after business footers.

Dummy fillers. data-insert-ptac-dummy-row-items controls whether PTAC-only pages can use dummy row items; set to n here to keep the demo clean. data-insert-dummy-row-while-format-table remains enabled to show full-height filler behavior if needed.
